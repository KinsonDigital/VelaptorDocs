If you've wanted a simple way to support the project financially, **[Ko-fi](https://ko-fi.com/kinsondigital)** is now available alongside GitHub Sponsors and Open Collective. Velaptor and the rest of the KinsonDigital projects are free and open source, and I want to keep them that way.

{/* truncate */}

## Why add Ko-fi?

Not everyone uses the same funding platform; while some prefer GitHub Sponsors or Open Collective, others simply want a straightforward way to support the project.

Ko-fi helps with exactly that.

It adds a frictionless funding option for the community, supporting both quick one-time donations and recurring long-term memberships.

## Why financial support matters

Open source is often described as "free," but it is never free to maintain.

Behind every release, bug fix, documentation update, sample project, and support conversation is real time, real effort, and real cost.
Financial support helps create room for that work to keep happening consistently instead of only when spare time allows. When open-source maintainers have reliable support, the entire project benefits.

Financial contributions help cover things like:

<ul style={{listStyleType: 'disc', paddingLeft: '2rem'}}>
    <li>Hosting and infrastructure costs</li>
    <li>Development tools and services</li>
    <li>Testing time and release preparation</li>
    <li>Writing and maintaining documentation</li>
    <li>Investigating bugs and improving usability</li>
</ul>

That support makes it easier for us to:
<ul style={{listStyleType: 'disc', paddingLeft: '2rem'}}>
    <li>Ship updates more consistently</li>
    <li>Respond to issues faster</li>
    <li>Improve documentation and onboarding</li>
    <li>Invest in polish, stability, and long-term maintenance</li>
    <li>Keep the project healthy without compromising its free and open nature</li>
</ul>

In short, funding helps turn good intentions into dependable progress.

## Every amount helps

You do not need to make a huge contribution to make a meaningful difference.
Every contribution, recurring or one-time, provides stability and proves this work is worth sustaining.
