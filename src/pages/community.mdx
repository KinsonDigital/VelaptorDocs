# Join Our Community

Velaptor is built by the community, for the community. Whether you write code, report bugs, or spread the word,
your involvement directly shapes the future of open-source game development for .NET.

---

## Contribute Your Skills

Whether you're a seasoned developer or just starting out, there's a place for you:

- **New to the project?** Start with <Url href="https://github.com/KinsonDigital/Velaptor/issues?q=is%3Aopen+is%3Aissue+label%3A%22good-first-issue%22" text="good first issues" openInNewTab={true}/> — curated tasks designed to get you up and running quickly.
- **Ready to dig in?** Pick up any <Url href="https://github.com/KinsonDigital/Velaptor/issues" text="open issue" openInNewTab={true}/> and submit a pull request with code improvements, bug fixes, or new features.
- **Found a bug or have an idea?** <Url href="https://github.com/KinsonDigital/Velaptor/issues/new/choose" text="Report it" openInNewTab={true}/> — your feedback directly shapes what gets built next.

Before submitting code, please read the <Url href="https://github.com/KinsonDigital/.github/blob/main/docs/CONTRIBUTING.md" text="contributing guidelines" openInNewTab={true}/> to make the review process smooth for everyone.

---

Every dollar goes directly toward keeping the project alive and growing.

- <Url href="https://github.com/sponsors/KinsonDigital" text="GitHub Sponsors" openInNewTab={true}/> — sponsor directly through GitHub.

Every contribution — code, feedback, or financial — makes a real difference. Join us and help make game development more accessible for everyone.
